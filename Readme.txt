Este proyecto se realizó siguiendo las instrucciones del siguiente curso de Udemi

www.udemy.com/course/crud-en-java-y-mysql-facil

Se tuvo como punto de culminación la integración de las funcionalidades de los botones "Eliminar", "Actualizar" y "Guardar"

Nota:Antes nos pondre en funcionamiento este proyecto tenga en cuenta de que debe estar correctamente referenciada la librería de Mysql para java
Ubicado en la siguiente dirección "src\Resources\mysql-connector-java-5.1.24-bin.jar"

El servidor al que se establece comunicación se ejecutó de manera local y usando la plataforma Xammp.
